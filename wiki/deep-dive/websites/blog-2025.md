### 1. 如何将 10G 11 万个文件的文件夹（微信存档）从一个 iPhone 传输到另一个 iPhone

**日期**: 2025-12-20  
**URL**: https://xzos.net/blog/iphone-large-folder-transfer/  
**分类**: 工具/iOS

#### 摘要

当传统方案（AirDrop、iCloud）无法处理大型文件夹时，使用 a-Shell + SSH + tar 的高效方案。

#### 技术要点

**问题场景**:
- AirDrop: 会无限等待"准备文件"
- iCloud: 直接报错，无法处理大量文件

**解决方案架构**:
```
iPhone (a-Shell) → SSH/tar → Linux 电脑 → SSH/tar → iPhone (a-Shell)
```

**关键命令**:
```bash
# 上传到电脑
tar czf - ./wx | ssh user@ip "cd ~/upload && tar xzf -"

# 从电脑下载
ssh -c aes128-ctr user@ip "cd ~/upload && tar czf - ." | tar xzf -
```

**性能**: Wi-Fi 内网 100Mbps，10GB 约 13-15 分钟

#### 技术深度
- a-Shell vs iSH 性能对比
- iSH 的 x86 模拟架构限制
- aes128-ctr 加密算法优化传输速度

#### 技术标签
`iOS` `a-Shell` `SSH` `tar` `大文件传输`

### 5. Azure Custom Vision: Use managed identity or other Azure Identity

**日期**: 2025-01-01  
**URL**: https://xzos.net/blog/azure-custom-vision-mi/  
**分类**: 开发/云服务

#### 摘要

介绍如何在 Azure Custom Vision 中使用 Managed Identity 替代 API Key 进行身份验证。

#### 技术要点

**工作原理**:
1. 使用 managed identity 登录 Resource Manager Endpoint
2. 获取 token 字符串
3. 使用 token 访问 CustomVisionTrainingClient

**C# 代码示例**:
```csharp
var tokenCredential = new DefaultAzureCredential();
var mercuryResourceUri = "https://cognitiveservices.azure.com";
var tokenRequestContext = new TokenRequestContext(new[] { $"{mercuryResourceUri}/.default" });
var token = tokenCredential.GetToken(tokenRequestContext).Token;

var trainingClient = new CustomVisionTrainingClient(new TokenCredentials(token))
{
    Endpoint = customVisionEndpoint,
};
```

#### 技术标签
`Azure` `Custom Vision` `Managed Identity` `C#` `身份验证`
