### 1. V2Ray 多协议多服务器情况使用 HAProxy 负载均衡

**日期**: 2019-11-23  
**URL**: https://xzos.net/blog/load-balancing-v2ray-with-haproxy-and-docker/  
**分类**: 网络/运维

#### 摘要

使用 Docker + HAProxy + V2Ray 实现多协议多服务器的负载均衡，自动维护可用节点列表。

#### 技术架构

```
用户 → V2Ray入口(Socks/HTTP) → HAProxy负载均衡 → 多个V2Ray出口容器 → 服务器
                                      ↓
                              健康检查(任意门+204)
```

**Docker Compose 配置**:
- v2ray (入口代理)
- haproxy (负载均衡)
- v2ray_out_X_tcp/quic (出口代理)

**HAProxy 健康检查**:
```
option httpchk GET http://google.com/generate_204 HTTP/1.1\r\nHost:\ google.com\r\n
server server1_name v2ray_out_0_tcp:1080 check port 80
```

**关键技术点**:
- V2Ray 任意门 (dokodemo-door) 转发健康检查
- Docker Compose 服务名自动路由
- mux 多路复用优化

#### 技术标签
`Docker` `HAProxy` `V2Ray` `负载均衡` `网络架构`
